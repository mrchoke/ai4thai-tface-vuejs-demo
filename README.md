# ai4thai-tface-vuejs-demo

DEMO สำหรับการใช้งาน AI For Thai T-Face API โดยใช้ Javascript

## ตัวอย่าง

- การเรียกใช้ T-Face API ด้วย Project Vue.js ผ่านทาง Fetch

## แก้ไข

```
src/compoments/TFace.vue
```

## Config

```
api_key:  ""
project_id: ""
```
